# mini-font
-A very tiny (probably tiniest) psf font-

CP437 character set (+ €, λ) with a fontsize of 4x5 pixels. "λ" is at 0xff (NBSP). Use [rw-psf by talamus](https://github.com/talamus/rw-psf) if needed.
